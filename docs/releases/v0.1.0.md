# v0.1.0 — The Spine

The first end-to-end milestone: an idea goes in, an autonomous agent pipeline
runs, and a complete multi-section website comes out — live, in the browser.

## Highlights

- **Autonomous pipeline.** Five agents collaborate over a typed
  `GenerationContext` blackboard: `CEO → Product Manager → Brand → Copywriting → Frontend`.
- **Durable & resumable.** Run state is checkpointed after every agent, so a
  killed run resumes from the last completed step instead of restarting.
- **Live cockpit.** Watch the agents work in real time over SSE, then see the
  generated site embedded in the page.
- **Token-driven renderer.** Owned section components (hero, features, pricing,
  FAQ, …) read color and type from CSS variables, so output is brand-themed
  rather than hardcoded.
- **Runs with or without an API key.** Live generation uses tiered Claude
  models; with no key (or no credit) it falls back to a deterministic offline
  generator so the product is always demoable.

## Try it

```bash
npm install
npm run dev   # then describe a startup at http://localhost:3000
```

Set `ANTHROPIC_API_KEY` in `.env.local` for live generation; otherwise it runs
in demo mode.
